# CCO-K

CCO-K는 비전공자 한국인이 Claude Code와 OMC를 최대한 쉽게 쓰도록 돕는 한국어 플러그인/환경 세트입니다.

핵심 목표는 간단합니다.
- 영어 명령어를 몰라도 시작할 수 있게 만들기.
- 처음 설치한 사람도 `/메뉴판` 하나로 길을 잃지 않게 만들기.
- Claude Code 기본 기능과 OMC 자동화 모드를 한국어 흐름으로 연결하기.

![CCO-K 메뉴판](docs/images/menu.png)

## 이 프로젝트가 해결하려는 문제

비전공자가 Claude Code를 처음 열면 보통 여기서 막힙니다.
- 무슨 명령어를 먼저 써야 하는지 모르겠음.
- OMC가 뭔지, 왜 필요한지 모르겠음.
- 설치는 했는데 실제로 뭘 입력해야 하는지 모르겠음.
- 영어 설명이 많아서 부담스러움.

CCO-K는 이 문제를 줄이기 위해 아래 흐름을 제공합니다.
- 설치.
- OMC 연결.
- `/메뉴판` 진입.
- `/길잡이` 또는 `/추천`으로 다음 행동 1개 추천.
- 실제 작업 명령 실행.

## 누구를 위한 도구인가요

이런 분에게 맞습니다.
- 코딩이 익숙하지 않은 한국어 사용자.
- Claude Code를 처음 쓰는 사람.
- OMC 자동화 기능을 써보고 싶지만 설정이 어려운 사람.
- 빠르게 화면 제작, 기능 구현, 에러 수정, 자동 검수까지 이어가고 싶은 사람.

## 가장 먼저 기억할 것

정말 이것만 기억하시면 됩니다.

```text
/메뉴판
```

설치와 연결이 끝난 뒤에는 `/메뉴판` 하나로 대부분 시작할 수 있습니다.

## 빠른 시작

자세한 설명은 `docs/빠른-시작.md` 에 정리되어 있습니다.

아주 짧게 요약하면 아래 순서입니다.

### 1단계. CCO-K 설치
Claude Code 안에서 아래를 실행합니다.

```text
/plugin marketplace add DinoEgg-code/claude-code-omc-korean
/plugin install cco-k@cco-k-marketplace
```

### 2단계. OMC 연결
설치가 끝나면 아래를 실행합니다.

```text
/ccok-setup
```

이 단계의 목표는 OMC를 현재 프로젝트와 연결하는 것입니다.

### 3단계. 시작
이제 아래만 입력하세요.

```text
/메뉴판
```

## 추천 시작 흐름

처음에는 이렇게 쓰는 걸 추천합니다.

### 아무것도 모르겠을 때
```text
/길잡이
```

### 지금 무엇을 해야 할지 추천받고 싶을 때
```text
/추천
```

### 내 아이디어를 AI가 잘 이해하는 문장으로 바꾸고 싶을 때
```text
/요청서 [아이디어]
```

## 처음엔 이것만 쓰세요

비전공자라면 먼저 `docs/핵심-명령-7개.md` 기준으로 시작하는 걸 추천합니다.

## 자주 쓰는 명령

### 만들기
- `/화면 [내용]`
- `/기능 [내용]`
- `/자동화 [내용]`

### 수정하기
- `/파일찾기 [내용]`
- `/설명`
- `/고쳐`
- `/정리`
- `/점검`
- `/체크리스트`

### 자동화 고급 모드
- `/자동개발 [내용]`
- `/끝까지 [내용]`
- `/초고속 [내용]`
- `/계획모드 [내용]`
- `/합의계획 [내용]`
- `/생각정리 [내용]`
- `/팀실행 [내용]`
- `/원인추적 [내용]`
- `/자동검수 [내용]`
- `/외부조언 [내용]`

## 비전공자용 추천 사용법

### 1. 먼저 목적을 짧게 말하기
예시.

```text
/기능 로그인 기능 만들어줘
```

```text
/화면 깔끔한 회원가입 화면 만들어줘
```

### 2. 막히면 다음 행동 하나만 추천받기
```text
/추천
```

### 3. 끝까지 맡기고 싶으면 자동화 모드 사용
```text
/자동개발 강의 사이트 MVP 만들어줘
```

## 주의할 점

- OMC 고급 기능은 먼저 `/ccok-setup` 으로 연결이 되어 있어야 안정적으로 작동합니다.
- 모르면 명령어를 여러 개 외우려고 하지 말고 `/메뉴판` 으로 다시 돌아오면 됩니다.
- 설명이 길게 필요하면 `/설명`.
- 수정 위치가 헷갈리면 `/파일찾기`.
- 에러가 나면 `/고쳐`.
