# claudecode-korean: 프로젝트 가이드 및 단축 명령어 (Skills & Commands)

이 프로젝트는 비전공자 1인 창업가를 위한 쉽고 안전하며, **가장 경제적인** 전용 환경입니다.
채팅창에 아래의 `/단축어`를 입력하면 필요한 전문 기능(Skill)만 불러와 실행합니다.

# 1. 첫 만남 및 스타일 (Onboarding & Style)
- **온보딩 가이드**: 새로운 세션이 시작되거나 사용자가 처음 인사하면 "무엇부터 할지 모르시겠다면 **/메뉴판**을 입력해 보세요!"라고 친절하게 안내한다.
- **시각적 UI (Box UI)**: 모든 안내나 메뉴판 출력 시 아래와 같은 박스 기호를 사용하여 터미널에서도 '앱'처럼 보이게 디자인한다.

## ⚡ 마법의 슬래시(/) 명령어 모음

### 💁‍♂️ 안내 데스크 (스킬 탐색)
- **`/메뉴판`** (또는 `/스킬`): 현재 장착된 모든 전문 스킬(.claude/skills/ 내 파일들)의 목록과 용도를 비전공자 눈높이에서 요약해서 보여줍니다.

- **`/추천`** (또는 `/recommend`): 현재 작업 상황을 분석하여 지금 가장 필요한 스킬을 클로드가 제안합니다.

### 🛠️ 개발 및 문제 해결 (핵심 스킬)

- **`/깃세팅`** (또는 `/git-init`): 프로젝트를 깃허브에 난생처음 연결할 때, 1번부터 4번까지 따라만 하면 되는 초기 설정 가이드를 보여줍니다.

- **`/저장`** (또는 `/save`): 코드를 다 짰을 때 입력하면, 요금 낭비 없이 깃허브에 안전하게 백업하는 3줄 명령어를 즉시 알려줍니다.

- **`/배포`** (또는 `/deploy`): 완성된 내 코드를 서버 비용 없이 인터넷 주소(링크)로 만들어 전 세계에 공개하는 방법을 알려줍니다.

- **`/기능 [내용]`** (또는 `/시작`): 새로운 기능을 기획하고 만듭니다. 필요한 폴더 구조와 코드를 순서대로 뚝딱 제안합니다.

- **`/화면 [내용]`** (또는 `/시작 ui-gen`): 예쁜 웹사이트 디자인과 UI 코드를 즉시 짜줍니다. (ui-gen 스킬 전용)

- **`/설명`** (또는 `/explain`): 현재 열려있는 파일이나 코드가 무슨 역할을 하는지 비전공자 눈높이에서 해설해 줍니다.

- **`/고쳐`** (또는 `/에러`, `/error`): 발생한 에러를 분석하고, "왜 발생했는지(쉬운 비유)" + "어떻게 고치는지(복붙용 코드)"를 제공합니다.

- **`/점검`** (또는 `/보안`, `/security`): 전체 프로젝트를 스캔하여 API 키 노출이나 보안 취약점을 철저히 점검하고 보고합니다.

- **`/정리`** (또는 `/refactor`, `/clean`): 지저분하게 작성된 코드를 비전공자가 읽기 쉽도록 단순화하고 한국어 주석을 달아 정리합니다.


### 💰 요금 폭탄 방지 (토큰 및 메모리 관리)
*(주의: 1인 창업가의 자본을 지키기 위해 아래 명령어를 자주 활용하세요!)*

- **`/청소`** (또는 `/초기화`, `/clear`): 완전히 새로운 작업을 시작할 때 입력합니다. 이전 대화의 기억을 싹 지워 불필요한 토큰 요금이 새어나가는 것을 막습니다. (무료, 즉시 초기화)

- **`/압축`** (또는 `/compact`): 대화가 너무 길어져서 렉이 걸리거나 요금이 걱정될 때 입력합니다. 지금까지의 핵심 내용만 요약하고 짐을 줄입니다.

- **`/비용`** (또는 `/cost`): 현재 세션에서 지금까지 얼마의 API 요금을 썼는지 계산해서 보여줍니다.

## 📂 개발 환경 및 클린 코드 가이드
- **유연한 기술 선택 (오마카세 모드)**: 무조건 특정 스택을 고집하지 않습니다. 작업의 크기와 목적을 분석하여 **가장 가볍고, 빠르고, 나중에 수정하기 쉬운 최신 기술(라이브러리 등)**을 클로드가 전문가의 시선으로 알아서 선택하고 제안합니다.
- **클린 코드 (Clean Code)**: 군더더기 없는 깔끔한 코드를 작성합니다. 비전공자 창업가가 나중에 코드를 보더라도 이해할 수 있도록, 복잡하게 꼬인 로직은 피하고 직관적인 구조를 최우선으로 합니다.
- **우선순위 최적화**: 너무 많은 MCP(외부 도구 서버)를 동시에 켜두면 요금이 급증하므로, 10개 미만의 필수 도구만 활성화하여 가볍고 경제적으로 유지합니다.
