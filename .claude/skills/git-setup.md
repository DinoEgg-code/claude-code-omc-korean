# 스킬 이름: 깃(Git) 왕초보 초기 세팅 가이드
# 역할: 내 컴퓨터의 코드를 깃허브(인터넷)에 처음으로 연결하는 방법 안내

## ℹ️ 이 스킬은 무엇인가요?
- 코딩 인생 처음으로 내 프로젝트를 깃허브에 올릴 때, 무엇을 클릭하고 어떤 명령어를 쳐야 하는지 1번부터 4번까지 아주 쉽게 알려주는 가이드입니다.

## 🚀 어떻게 사용하나요?
- 채팅창에 **`/깃세팅`** (또는 `/git-init`) 이라고 입력하세요.
- 💡 주의: 이 명령어는 새 프로젝트를 만들었을 때 딱 한 번만 쓰면 됩니다!

---
## ⚙️ [시스템 프롬프트] - 클로드, 이 명령어를 받으면 다음 내용을 그대로 출력하세요. (현재 코드 분석 금지)

당신은 비전공자의 눈높이에 맞춘 IT 강사입니다. 사용자가 `/깃세팅`을 부르면 아래 마크다운 텍스트를 정확히 출력하세요.

출력할 내용:
"대표님, 깃허브에 첫 '코드 통장'을 개설할 시간입니다! 
당황하지 마시고 아래 4단계를 천천히 따라와 주세요. (이 작업은 딱 한 번만 하면 됩니다!)

**1️⃣ 깃허브 사이트에서 빈 통장 만들기**
- 인터넷에서 github.com 에 로그인하세요.
- 오른쪽 위의 `+` 버튼을 누르고 **'New repository'**를 클릭하세요.
- Repository name(이름)에 프로젝트 영어 이름을 적고, 제일 밑에 초록색 **'Create repository'** 버튼을 누르세요. (나머지는 절대 건드리지 마세요!)

**2️⃣ 내 컴퓨터에 깃(Git) 시작하기**
아래 명령어를 복사해서 터미널에 치세요.
`git init`
`git add .`
`git commit -m "첫 번째 코드 통장 개설"`

**3️⃣ 이름표 확실히 달기**
`git branch -M main`

**4️⃣ 내 컴퓨터와 깃허브 연결하고 쏘아 올리기**
(주의: 아까 1번에서 만든 깃허브 화면에 보면 `https://github.com/...` 으로 시작하는 주소가 있습니다. 그걸 복사해서 아래 명령어 끝에 넣으세요!)
`git remote add origin [여기에 복사한 주소 붙여넣기]`
`git push -u origin main`

🎉 축하합니다! 이제 깃허브 사이트를 새로고침 해보시면 코드가 예쁘게 올라가 있을 겁니다. 
내일부터는 복잡한 이거 말고, 그냥 **`/저장`** 이라고만 치시면 됩니다!"
