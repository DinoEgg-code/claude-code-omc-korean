# CCO-K 빠른 시작

이 문서는 비전공자가 CCO-K를 처음 설치하고 바로 쓰기 위한 가장 쉬운 안내서입니다.

## 목표

아래 3단계만 끝내면 됩니다.
1. CCO-K 설치.
2. OMC 연결.
3. `/메뉴판` 으로 시작.

## 1단계. CCO-K 설치

Claude Code 안에서 아래 두 줄을 실행하세요.

```text
/plugin marketplace add DinoEgg-code/claude-code-omc-korean
/plugin install cco-k@cco-k-marketplace
```

설치가 끝나면 다음 단계로 넘어갑니다.

## 2단계. OMC 연결

이제 아래를 실행하세요.

```text
/ccok-setup
```

이 단계는 OMC 자동화 기능을 현재 환경에 연결하기 위한 단계입니다.

OMC가 아직 설치되지 않았으면 CCO-K가 필요한 다음 행동을 안내합니다.

## 3단계. 시작

이제 아래만 입력하면 됩니다.

```text
/메뉴판
```

## 처음엔 이렇게 쓰세요

### 뭘 해야 할지 모르겠을 때
```text
/길잡이
```

### 다음 행동 하나만 추천받고 싶을 때
```text
/추천
```

### 내 생각을 AI가 잘 알아듣는 요청으로 바꾸고 싶을 때
```text
/요청서 쇼핑몰 첫 화면 만들고 싶어
```

### 화면부터 보고 싶을 때
```text
/화면 깔끔한 랜딩페이지 만들어줘
```

### 기능을 만들고 싶을 때
```text
/기능 이메일 로그인 기능 만들어줘
```

### 끝까지 자동으로 맡기고 싶을 때
```text
/자동개발 강의 사이트 MVP 만들어줘
```

## 막히면 이렇게 하세요

- 다시 메뉴가 필요하면 `/메뉴판`
- 지금 뭘 해야 할지 모르겠으면 `/추천`
- 수정 위치를 모르겠으면 `/파일찾기`
- 에러가 나면 `/고쳐`
- 설명이 필요하면 `/설명`

## 꼭 기억할 것

정말 헷갈리면 다시 이 한 줄만 입력하면 됩니다.

```text
/메뉴판
```
