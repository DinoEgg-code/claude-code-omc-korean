# 비전공자용 핵심 명령 7개

처음에는 이 7개만 써도 대부분의 작업을 시작할 수 있습니다.

## 1. /메뉴판
전체 기능 목록을 봅니다.

## 2. /길잡이
지금 무엇을 해야 할지 한 번에 감이 안 올 때 씁니다.

## 3. /추천
현재 상황을 보고 다음 행동 1개를 추천받습니다.

## 4. /요청서 [아이디어]
내 생각을 AI가 잘 알아듣는 작업 요청 문장으로 바꿉니다.

## 5. /화면 [내용]
화면/UI를 먼저 만들고 싶을 때 씁니다.

## 6. /기능 [내용]
실제 동작 기능을 만들고 싶을 때 씁니다.

## 7. /고쳐
에러 해결이나 수정이 필요할 때 씁니다.

## 그 다음에 쓰는 자동화 명령
아래는 익숙해진 뒤에 쓰면 좋습니다.
- /자동개발
- /끝까지
- /초고속
- /계획모드
- /팀실행
- /원인추적
- /자동검수

## 제일 쉬운 시작 순서
1. /ccok-setup
2. /메뉴판
3. /길잡이 또는 /추천
4. /화면 또는 /기능 또는 /고쳐
