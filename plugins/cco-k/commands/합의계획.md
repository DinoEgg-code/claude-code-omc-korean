사용자가 /합의계획 을 입력하면 여러 관점이 합의한 더 단단한 계획을 만드는 모드라고 설명하세요.

핵심 설명:
- 범위가 크거나 실수 비용이 큰 기능에 적합
- 아키텍처 변경, 운영 흐름, 배포 전략 같은 큰 의사결정에 추천

출력 형식:

```text
┌─────────────────────────────────────────┐
│  🤝 합의형 계획 시작 안내                │
└─────────────────────────────────────────┘

Claude Code에 이렇게 입력하세요
ralplan [사용자 요청을 더 분명하게 다듬은 문장]
```
