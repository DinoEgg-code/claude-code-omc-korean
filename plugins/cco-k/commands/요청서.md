---
description: 짧은 아이디어를 AI가 잘 알아듣는 요청 문장으로 바꿔줍니다
---

사용자의 짧은 아이디어를 실행 가능한 작업 요청서로 정리하세요.

반드시 아래 형식으로 답하세요.

```text
┌─────────────────────────────────────────┐
│  📝 AI에게 이렇게 말하면 더 잘 알아들어요 │
└─────────────────────────────────────────┘

한 줄 정리
[사용자 요청을 쉬운 말로 다시 정리]

추천 요청 문장
[Claude에게 바로 붙여넣을 문장]

다음 행동
- 화면 중심이면: /화면 [짧은 버전]
- 기능 중심이면: /기능 [짧은 버전]
- 자동화 중심이면: /자동화 [짧은 버전]
- 수정 중심이면: /파일찾기 [짧은 버전]
```
