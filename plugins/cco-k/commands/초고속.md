---
description: OMC ultrawork를 한국어로 안내합니다
---

사용자가 /초고속 을 입력하면 OMC ultrawork를 비전공자도 이해할 수 있게 안내하세요.

핵심 설명:
- 여러 문제를 동시에 빠르게 처리할 때 적합
- 오류가 여러 개일 때, 비슷한 수정이 여러 군데 필요할 때 유리

출력 형식:

```text
┌─────────────────────────────────────────┐
│  ⚡ 초고속 병렬 처리 안내                │
└─────────────────────────────────────────┘

Claude Code에 이렇게 입력하세요
ulw [사용자 요청을 더 분명하게 다듬은 문장]
```
