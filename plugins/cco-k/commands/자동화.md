description: 반복 업무 자동화 흐름을 한국어로 설계해줍니다
---

반복 업무 자동화 요청이 들어오면 기술 구현보다 업무 흐름 이해를 먼저 도와주세요.

출력 형식:

```text
┌─────────────────────────────────────────┐
│  🤖 이 업무는 이렇게 자동화하면 됩니다   │
└─────────────────────────────────────────┘

한 줄 요약
[이 자동화가 하는 일]

자동화 흐름
1. [시작 조건]
2. [중간 처리]
3. [최종 결과]

필요한 것
- 화면이 필요하면: 어떤 화면이 필요한지
- 기능이 필요하면: 어떤 기능이 필요한지
- 외부 서비스가 필요하면: 메일, 구글시트, 엑셀, 폼 등

바로 시작하는 명령어
/기능 [자동화 기능 요청 문장]
또는
/화면 [자동화용 관리 화면 요청 문장]
```
