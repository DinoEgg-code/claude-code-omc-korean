```text
처음이시라면 아래 3단계만 따라오시면 됩니다.

1단계. CCO-K 설치
/plugin marketplace add DinoEgg-code/claude-code-omc-korean
/plugin install cco-k@cco-k-marketplace

2단계. OMC 연결
/ccok-setup

3단계. 시작
/메뉴판

설치가 끝난 뒤에는 /메뉴판 하나만 기억하시면 됩니다.
```
