description: GitHub 첫 연결 방법을 한국어로 안내합니다
---

아래 내용을 그대로 보여주세요.

```text
대표님, GitHub에 첫 코드 통장을 개설합니다. 이 작업은 한 번만 하면 됩니다.

1️⃣ github.com 로그인 → 오른쪽 위 + → New repository
   이름 입력 → Create repository

2️⃣ 터미널에 아래 3줄 하나씩 입력:
   git init
   git add .
   git commit -m "첫 번째 코드 통장 개설"

3️⃣ 아래 명령 입력:
   git branch -M main

4️⃣ GitHub 저장소 주소를 복사한 뒤 아래처럼 입력:
   git remote add origin [복사한 주소]
   git push -u origin main

완료되면 다음부터는 /저장 만 쓰시면 됩니다.
```
