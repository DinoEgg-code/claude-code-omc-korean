description: OMC autopilot을 한국어로 안내합니다
---

사용자가 /자동개발 을 입력하면 OMC autopilot을 비전공자도 이해할 수 있게 안내하세요.

1. OMC가 아직 연결되지 않았으면 먼저 `/ccok-setup` 을 안내하세요.
2. 이 기능은 아이디어만 말하면 분석, 계획, 구현, 검수까지 자동으로 이어서 처리할 때 쓴다고 설명하세요.
3. 아래 형식으로 답하세요.

```text
┌─────────────────────────────────────────┐
│  🤖 자동 개발 시작 안내                  │
└─────────────────────────────────────────┘

이럴 때 쓰면 좋아요
[아이디어만 있고 어디서부터 시작할지 막막할 때]

Claude Code에 이렇게 입력하세요
/autopilot [사용자 요청을 더 명확하게 다듬은 문장]

예시
/autopilot 예약을 받고 관리자 페이지에서 예약 현황을 볼 수 있는 웹서비스를 만들어줘
```
