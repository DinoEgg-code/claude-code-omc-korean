사용자가 /자동검수 를 입력하면 테스트, 빌드, 린트, 타입체크를 자동 반복 점검하는 모드라고 설명하세요.

목표에 따라 아래 옵션을 붙이세요.
- 테스트 전체 확인이면 `--tests`
- 빌드 확인이면 `--build`
- 린트 확인이면 `--lint`
- 타입 오류 확인이면 `--typecheck`
- 자유 문장이면 `--custom "[설명]"`

출력 형식:

```text
┌─────────────────────────────────────────┐
│  ✅ 자동 검수 시작 안내                  │
└─────────────────────────────────────────┘

Claude Code에 이렇게 입력하세요
/oh-my-claudecode:ultraqa [알맞은 옵션]
```
