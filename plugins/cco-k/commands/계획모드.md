사용자가 /계획모드 를 입력하면 먼저 질문하며 계획부터 정리하는 모드라고 설명하세요.

핵심 설명:
- 바로 코드부터 짜기보다 계획을 먼저 정리하고 싶을 때 적합
- 로그인, 결제, 데이터 구조, API 설계처럼 첫 결정이 중요한 작업에 추천

출력 형식:

```text
┌─────────────────────────────────────────┐
│  🗺️ 계획 인터뷰 시작 안내                │
└─────────────────────────────────────────┘

Claude Code에 이렇게 입력하세요
/oh-my-claudecode:plan [사용자 요청을 더 분명하게 다듬은 문장]
```
