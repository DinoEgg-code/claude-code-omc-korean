description: 저장 전, 배포 전, 오류 수정 후 확인할 항목을 짧게 보여줍니다
---

현재 상황에 맞는 확인 항목만 3~5개로 짧게 보여주세요.

상황별 기준:
- 배포 직전
- 코드 수정 직후
- 에러 해결 직후
- 상황 불분명

출력 형식:

```text
┌─────────────────────────────────────────┐
│  ✅ 지금 확인하면 좋은 항목              │
└─────────────────────────────────────────┘

- [확인 항목 1]
- [확인 항목 2]
- [확인 항목 3]
- [다음 명령어]
```
