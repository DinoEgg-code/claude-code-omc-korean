사용자가 /팀실행 을 입력하면 여러 전문가가 역할을 나눠서 큰 작업을 처리하는 모드라고 설명하세요.

핵심 설명:
- 작업이 크고 역할을 나눠야 할 때 적합
- 프론트/백엔드/검수처럼 여러 역할이 필요한 기능에 추천

출력 형식:

```text
┌─────────────────────────────────────────┐
│  👥 팀 실행 시작 안내                    │
└─────────────────────────────────────────┘

Claude Code에 이렇게 입력하세요
team 3:executor [사용자 요청을 더 분명하게 다듬은 문장]
```
