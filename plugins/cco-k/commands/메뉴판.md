```text
┌─────────────────────────────────────────────────────────────┐
│          🇰🇷 CCO-K 메뉴판 (이것만 기억하세요)               │
└─────────────────────────────────────────────────────────────┘

대표님, 처음에는 아래 순서만 기억하시면 됩니다.
1) /ccok-setup
2) /메뉴판
3) 필요한 명령 선택

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
초급 시작 명령
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
/길잡이            → 완전 처음일 때, 지금 뭘 해야 할지 1개 추천
/추천              → 이미 작업 중일 때, 다음 행동 1개 추천
/요청서 [아이디어]  → 내 생각을 AI가 잘 알아듣는 요청 문장으로 정리
/화면 [내용]       → 예쁜 웹 화면 코드 생성
/기능 [내용]       → 새 기능 기획 + 코드 작성
/고쳐              → 에러 수정과 해결 코드 안내

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
작업/마무리 명령
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
/파일찾기 [내용]   → 어느 파일을 열어야 할지 찾기
/설명              → 현재 코드 역할 해설
/정리              → 복잡한 코드 정리
/점검              → 보안 점검
/체크리스트        → 저장 전 / 배포 전 확인
/깃세팅            → GitHub 첫 연결
/저장              → Git & GitHub 백업
/배포              → Vercel 배포

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
고급 자동화 명령 (OMC)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
/ccok-setup        → OMC 설치와 연결을 먼저 도와줌
/자동개발 [내용]   → 아이디어부터 구현까지 자동 진행
/끝까지 [내용]     → 완료될 때까지 계속 밀어붙이기
/초고속 [내용]     → 여러 작업을 동시에 빠르게 처리
/계획모드 [내용]   → 질문하며 계획부터 정리
/합의계획 [내용]   → 여러 관점으로 더 깊게 합의된 계획 만들기
/생각정리 [내용]   → 모호한 아이디어를 질문으로 선명하게 만들기
/팀실행 [내용]     → 여러 전문가가 역할을 나눠 작업
/원인추적 [문제]   → 왜 문제가 생겼는지 깊게 추적
/딥분석 [문제]     → 구조/성능/반복 장애를 깊게 분석
/자동검수 [목표]   → 테스트/빌드/린트를 자동 반복 점검
/외부조언 [질문]   → 다른 AI 관점까지 참고

헷갈리기 쉬운 차이
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
/길잡이  → 완전 처음 시작할 때

/고쳐    → 빨리 수정

/딥분석  → 구조/성능/반복 문제 깊게 분석

/끝장내기 는 /끝까지 와 같은 뜻입니다.
/팀플레이 는 /팀실행 과 같은 뜻입니다.

┌─────────────────────────────────────────────────────────────┐
│  💡 헷갈리면 다시 /메뉴판 을 입력하시면 됩니다.             │
```
