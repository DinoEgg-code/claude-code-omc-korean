---

사용자에게 아래 메뉴판을 그대로 보여주세요.

```text
┌─────────────────────────────────────────────────────────────┐
│          🇰🇷 CCO-K 메뉴판 (이것만 기억하세요)               │
└─────────────────────────────────────────────────────────────┘

대표님, 초보자라면 /메뉴판 하나만 기억하시면 됩니다.
아래에서 지금 필요한 기능을 골라 그대로 입력해 보세요.
이 메뉴판에는 기본 한국어 기능과 OMC 자동화 기능이 함께 들어 있습니다.

🚀 지금 뭘 해야 할지 모르겠을 때
  /길잡이            → 지금 상황을 말하면 딱 맞는 다음 명령어 1개 추천
  /추천              → 현재 작업 흐름을 보고 다음 행동 추천
  /요청서 [아이디어]  → 내 생각을 AI가 잘 알아듣는 요청 문장으로 정리

🧩 만들고 싶을 때
  /화면 [내용]       → 예쁜 웹 화면 코드 즉시 생성
  /기능 [내용]       → 새 기능 기획 + 코드 작성
  /자동화 [내용]     → 반복 업무 자동화 흐름 설계

🛠️ 수정하고 싶을 때
  /파일찾기 [내용]   → 어느 파일을 열어야 할지 찾기
  /설명              → 현재 코드가 무슨 역할인지 쉬운 말로 해설
  /고쳐              → 에러 원인 + 해결 코드 제공
  /정리              → 복잡한 코드를 읽기 쉽게 정리
  /점검              → 보안 취약점 스캔
  /체크리스트        → 저장 전 / 배포 전 확인 항목 보기

🤖 자동 고급 기능 (OMC 연동)
  /ccok-setup        → OMC 설치와 연결을 먼저 도와줌
  /자동개발 [내용]   → 아이디어부터 구현까지 자동 진행 (OMC autopilot)
  /끝까지 [내용]     → 끝날 때까지 계속 밀어붙이기 (OMC ralph)
  /초고속 [내용]     → 여러 작업을 동시에 빠르게 처리 (OMC ulw)
  /계획모드 [내용]   → 먼저 질문하며 계획부터 정리 (OMC plan)
  /합의계획 [내용]   → 여러 관점으로 더 깊게 합의된 계획 만들기 (OMC ralplan)
  /생각정리 [내용]   → 모호한 아이디어를 질문으로 선명하게 만들기 (OMC deep-interview)
  /팀실행 [내용]     → 여러 전문가 팀이 나눠서 작업하기 (OMC team)
  /원인추적 [문제]   → 왜 문제가 생겼는지 깊게 추적 (OMC trace)
  /자동검수 [목표]   → 테스트/빌드/린트를 자동 반복 점검 (OMC ultraqa)
  /외부조언 [질문]   → 다른 AI 관점까지 참고 (OMC ask)

📁 저장하고 공개할 때
  /깃세팅            → GitHub에 처음 연결할 때 (첫 작업시에만 1회)
  /저장              → 코드를 Git & GitHub에 백업
  /배포              → Vercel로 무료 인터넷 배포

💰 요금 관리
  /청소              → 대화 기억 초기화
  /압축              → 긴 대화 요약
  /비용              → 현재 세션 사용 요금 확인

┌─────────────────────────────────────────────────────────────┐
│  💡 OMC 자동 기능을 쓰려면 먼저 /ccok-setup 을 실행하세요. │
└─────────────────────────────────────────────────────────────┘
```
