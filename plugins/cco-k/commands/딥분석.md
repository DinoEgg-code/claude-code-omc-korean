description: OMC analyze를 한국어로 안내합니다
---

사용자가 /딥분석 을 입력하면 복잡한 이슈를 구조적으로 깊게 분석하는 모드라고 설명하세요.

핵심 설명:
- 단순 오류 수정이 아니라 원인 구조를 파고들 때 유용
- 메모리 누수, 성능 저하, 반복 장애 같은 문제에 적합

출력 형식:

```text
┌─────────────────────────────────────────┐
│  🔍 딥 분석 시작 안내                    │
└─────────────────────────────────────────┘

Claude Code에 이렇게 입력하세요
/oh-my-claudecode:analyze "[사용자 문제를 명확하게 다듬은 문장]"
```

주의사항:
- 단순 문법 오류는 `/고쳐` 가 더 빠를 수 있다고 알려주세요.
- 원인이 애매하고 반복되는 문제는 `/원인추적` 이 먼저 맞을 수 있다고 알려주세요.
- `/딥분석` 은 성능, 구조, 반복 장애처럼 문제 자체를 깊게 볼 때 가장 적합하다고 안내하세요.
